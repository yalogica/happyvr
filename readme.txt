== Installation ==
Setup is easy and takes only a few minutes.

1. Download the plugin, unpack it and upload the **[happyvr]** folder to your **wp-content/plugins/** directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Create a new post or page
4. Add the HappyVR Block from the block category called "Widgets" in the Gutenberg editor.
5. You can change block settings from the right-side settings sidebar.
You‚Äôre done. Enjoy.
